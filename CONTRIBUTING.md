# Contributing to CommonIntents-144

Thank you for your interest in the INTENT-7/CAPABILITY-13 protocol family.

## What We Welcome

- Improvements to protocol specification text (typos, unclear wording, logical gaps)
- Proposals for new `x-` extension fields
- Conformance test cases
- Documentation translations
- Implementation experience feedback (what you've built based on the protocols)

## What We Do Not Currently Accept

- Fundamental changes to the core philosophy of the protocols (trust minimization, on-demand activation, syntactic standard)
  — these are the constitution of the protocols, not within the scope of RFC discussion

## How to Contribute

1. Open an Issue for discussion before submitting a Pull Request
2. Link your PR to the corresponding Issue
3. Keep PRs small and focused; one PR, one thing
4. For protocol text changes, please update both the English and Chinese versions

## Code of Conduct

- Discuss the matter at hand; no personal attacks
- Respect the design philosophy of the protocols
- Maintainers reserve the right of final discretion
